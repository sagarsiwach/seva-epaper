## If Something Goes Wrong

### Error: "Port 3000 is in use"
- It's fine! The app will use port 3001
- Open: `http://localhost:3001` instead

### App is slow
- Close other browser tabs
- Clear cache (Ctrl+Shift+Delete)
- Reload (F5)

### App won't start
```bash
# Reinstall and rebuild
npm install
npm run build
npm run dev
```

### PDFs not showing
- Check console (F12 in browser)
- Look for errors in red text
- Verify PDFs exist in: `public/editions/`
